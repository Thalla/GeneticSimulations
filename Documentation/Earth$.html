  13.  def init(aaRSnumb: Int, possibleAARSnumb: Int, aaRSlength: Int, initAA: Vector[AA.AA]): (Cell, Vector[AARS])

    creates a start cell, each cell is one generation

    creates a start cell, each cell is one generation

    aaRSnumb

    Number of how many different aaRS proteins exist initially. Must be same or lower as numb of possible twoTuples because aaRSnumb defines the length of the code table. Therefore aaRSnumb isn't allowed to be bigger than max numb of codons/tuples. It must be lower than initAA.length/nunmber of existing amino acids. Otherwise the same AA is coded by various aaRS. aaRSnumb defines the length of the code table. TODO: allow codons to be translated by multiple aaRS (V1: to multiple AA (-> chaos?), V2: to always the same AA) -> changes in gene table and aaRSs creation

    aaRSlength

    The number of amino acids an aaRS consists of. TODO: aaRS can have varying lengths

    initAA

    initially existing and used amino acids, must be lower or same as maxTupleNumb(there are 16 twoTuple) TODO: allow having new/non proteinogenic amino acids

    returns

    start cell and all aaRS that can exist
